# Codex Visual Checklist

Before calling the build finished, verify:

- [ ] No left sidebar exists.
- [ ] The bottom quest path is the only chapter navigation.
- [ ] Four women scientist characters are visible and consistent.
- [ ] The main scene is layered rather than flattened.
- [ ] Dialogue remains editable text.
- [ ] Current chapter glows pink/purple.
- [ ] Completed chapters show green checks.
- [ ] Future chapters are dimmed or locked.
- [ ] NEXT advances and completed nodes go backward.
- [ ] The info icon opens a concise popup.
- [ ] The game fits a 1280×720 projector view without scrolling.
- [ ] Author Mode can export valid JSON.
- [ ] The build works after page refresh on GitHub Pages.
- [ ] Screenshots were compared with the primary reference.
