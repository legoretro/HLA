# Science Editing Guide

As you learn more, edit the educational wording in:

`hla-quest-project/src/content/chapters.json`

Use short dialogue for what you say aloud. Use the info popup for one optional clarification.

Keep each chapter to:
- one central concept
- one short dialogue block
- one optional info popup
- one visual or micro-interaction

Do not place scientific text directly into the PNG backgrounds. That makes later corrections difficult.

For solid-organ compatibility, preserve this distinction:

- DNA-based HLA typing identifies HLA alleles.
- Antibody screening detects anti-HLA antibodies.
- Crossmatching tests donor–recipient immune reactivity.
- The transplant team combines all results with clinical information.
